1. DB에 파일을 저장하고 싶으면 binary 모드로 저장해야 한다. (오라클 blob)
   우리나라는 파일 폴더(서버)를 만들고 이곳에 파일을 저장하고,
   DB에는 파일 위치와 파일명만 저장한다.

   tb_board에 파일명 필드를 추가하는게 나을까?
   프로젝트가 커지거나 파일 업로드 건수가 많을 경우에는 별도의 파일 관리 테이블을 구성한다.
   게시판 식별값, 파일 식별값 - primary key
   게시판 식별값(foreign key), 파일 식별값(primary key)

   DB는 순차검색, 첫 번째부터 찾을 때까지 검색한다.
   색인순차검색(인덱스를 만들어서 검색한다.) - 데이터를 정렬해서 색인표를 만든다.
   ㄱ~ㄷ : 1~100
   ㄹ~ㅁ : 101~200
   primary key로 지정된 필드는 자동 색인순차검색을 한다.
   순차검색 시 병행처리 - 한 번에 여러개씩 팍팍 읽는다.
   색인검색 시 - 한 번에 하나씩 처리, 더 느릴 때도 있음
